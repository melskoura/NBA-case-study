# NBA-case-study
# 🏀 NBA Data Analysis Project

## 📌 Overview

This project analyzes NBA games and player statistics to identify performance trends and key factors influencing game outcomes.

## 🛠️ Tools & Technologies

* Python (pandas, matplotlib, seaborn, plotly)
* SQL
* Tableau

## 🔍 Key Analysis

* Data cleaning and preprocessing using pandas
* Exploratory Data Analysis (EDA)
* Scoring trends over time
* Player performance and efficiency metrics
* Relationship between assists and scoring

## 📊 Key Insights

* Teams with higher assist rates tend to have better offensive efficiency
* Scoring trends indicate a more offensive-focused modern game
* Strong correlation between ball movement and team performance

## 📈 Dashboard

Interactive Tableau dashboard showcasing:

* Team performance
* Scoring trends
* Player efficiency

## 🚀 Project Structure

* `data/` → raw dataset
* `notebooks/` → analysis code
* `visuals/` → charts and images
